# Sony PlayStation Portable Data Storage Guide

## Console Overview
- **Release Dates**: Japan (December 12, 2004), North America (March 24, 2005), Europe (September 1, 2005)
- **Active Years**: 2004-2014
- **Regional Variants**:
  - PSP-1000 (Fat): Original model with infrared
  - PSP-2000/3000 (Slim): Lighter, video output capability
  - PSP Go (PSP-N1000): Digital-only, sliding screen

## Storage Media
- **Game Media**: Universal Media Disc (UMD) - 1.8GB capacity
- **Save Storage**: Memory Stick Pro Duo (32MB to 32GB)
- **Digital Distribution**: PlayStation Store downloads
- **UMD Format**: Proprietary optical disc in protective case

## Archival Storage
### Recommended Formats
- **.iso**: UMD disc image format
- **.cso**: Compressed ISO format
- **.pbp**: PlayStation Portable Boot format (PSN games)

### Best Practices
- Preserve UMD images at full quality
- Include save data and system files
- Archive PSN exclusive titles
- Document firmware requirements
- Preserve homebrew and custom firmware

## Emulation Storage
### Recommended Formats
- **.iso**: Best compatibility with PPSSPP
- **.cso**: Space-efficient compressed format
- **.pbp**: Required for PSN titles

### Considerations
- PPSSPP emulator offers broad compatibility
- Save data stored in separate files
- Custom firmware enables additional features
- Total library size: ~400GB uncompressed

## Digital Storage Considerations
- **Space Requirements**: Moderate - manageable collection size
- **Backup Strategy**: Important due to PSN store closure
- **Organization**: Separate UMD dumps from PSN titles
- **Metadata**: Use scene databases for verification
- **Compression**: CSO offers good space savings
- **Emulation**: Well-supported through PPSSPP
