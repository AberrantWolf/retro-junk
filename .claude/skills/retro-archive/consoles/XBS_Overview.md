# Microsoft Xbox Series X/S Data Storage Guide

## Console Overview
- **Release Date**: November 10, 2020 (worldwide)
- **Active Years**: 2020-present
- **Regional Variants**:
  - Xbox Series X: High-end model with disc drive
  - Xbox Series S: Digital-only, lower-spec model
  - Minimal regional hardware differences

## Storage Media
- **Disc Capacity**: 100GB Ultra HD Blu-ray (Series X only)
- **Internal Storage**: Custom NVMe SSD (1TB Series X, 512GB Series S)
- **Expandable Storage**: Proprietary expansion cards
- **Digital Distribution**: Microsoft Store (primary method)

## Archival Storage
### Current Limitations
- **Emulation**: No emulators exist
- **Dumping**: No homebrew/dumping tools available
- **Encryption**: Advanced system-level encryption
- **Legal Issues**: Current generation console

### Future Considerations
- Game sizes often 100GB+
- Smart Delivery system complicates preservation
- Quick Resume feature preservation
- System software evolution important

## Digital Storage Considerations
- **Space Requirements**: Enormous - 100GB+ per game
- **Backup Strategy**: Focus on documentation and legal preservation
- **Legal Status**: Current generation - preservation legally complex
- **Future Planning**: Standards and methods undetermined
- **Priority**: Document ecosystem and available content
