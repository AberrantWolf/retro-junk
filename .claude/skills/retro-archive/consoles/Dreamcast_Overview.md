# Sega Dreamcast Data Storage Guide

## Console Overview
- **Release Dates**: Japan (November 27, 1998), North America (September 9, 1999), Europe (October 14, 1999)
- **Active Years**: 1998-2001 (officially discontinued 2001)
- **Regional Variants**:
  - Minimal hardware differences between regions
  - Different case colors and limited editions
  - Built-in modem for online play

## Storage Media
- **Disc Capacity**: 1.2GB GD-ROM (Gigabyte Disc)
- **Disc Format**: Proprietary high-density CD format
- **Save Storage**: Visual Memory Unit (VMU) - 128KB
- **Online Features**: Built-in 56K modem, broadband adapter

## Archival Storage
### Recommended Formats
- **.gdi**: Gigabyte Disc Image (complete preservation)
- **.cdi**: DiscJuggler format (for burning to CD-R)
- **.chd**: Compressed format for space efficiency

### Best Practices
- Use GDI format for complete preservation
- Include VMU save data and mini-games
- Preserve online content when possible
- Archive homebrew and indie releases
- Document disc pressing variations

## Emulation Storage
### Recommended Formats
- **.gdi**: Best accuracy and compatibility
- **.chd**: Space-efficient compressed format
- **.cdi**: Legacy format, limited use

### Considerations
- Excellent emulation with Flycast/Redream
- VMU data stored separately
- Online features not preserved in emulation
- Some games require specific emulator settings
- Total library size: ~800GB uncompressed

## Digital Storage Considerations
- **Space Requirements**: Moderate - manageable collection
- **Backup Strategy**: Important for complete preservation
- **Organization**: Separate by region and release type
- **Metadata**: Use Redump database for GDI verification
- **Compression**: CHD recommended for long-term storage
- **Emulation**: Excellent preservation through modern emulators
