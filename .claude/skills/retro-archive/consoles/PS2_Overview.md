# Sony PlayStation 2 Data Storage Guide

## Console Overview
- **Release Dates**: Japan (March 4, 2000), North America (October 26, 2000), Europe (November 24, 2000)
- **Active Years**: 2000-2013 (longest-running console)
- **Regional Variants**:
  - Fat models (SCPH-10000 to SCPH-50000 series)
  - Slim models (SCPH-70000 to SCPH-90000 series)
  - Regional lockout and different video standards

## Storage Media
- **Disc Capacity**: 4.7GB DVD-5 (single layer), 8.5GB DVD-9 (dual layer)
- **Disc Formats**: DVD-ROM, CD-ROM (PS1 backwards compatibility)
- **Save Storage**: Memory Cards (8MB), USB drives, internal HDD (Network Adapter)
- **Special Formats**: Blue-backed discs for certain games

## Archival Storage
### Recommended Formats
- **.iso**: Standard DVD image format
- **.bin/.cue**: For games with special audio tracks
- **.chd**: Compressed format for space efficiency

### Best Practices
- Preserve complete disc images including video content
- Include memory card saves and system data
- Document disc layer information (single/dual)
- Archive network adapter and HDD content
- Preserve regional BIOS versions

## Emulation Storage
### Recommended Formats
- **.iso**: Best compatibility with PCSX2
- **.chd**: Space-efficient alternative
- **.cso**: Compressed ISO format

### Considerations
- Large file sizes require substantial storage
- BIOS files essential for emulation
- Memory card images stored separately
- Network features require special setup
- Total library size: ~8TB+ uncompressed

## ROM Format Reference
See [PS2.md](../formats/PS2.md) for detailed header format, checksum algorithms, and detection method.

## Digital Storage Considerations
- **Space Requirements**: Very high - multi-TB storage needed
- **Backup Strategy**: Critical due to large collection size
- **Organization**: Separate by region and disc type
- **Metadata**: Use Redump database for verification
- **Compression**: CHD recommended for long-term storage
- **Performance**: Fast storage recommended for optimal emulation
