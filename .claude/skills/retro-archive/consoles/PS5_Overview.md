# Sony PlayStation 5 Data Storage Guide

## Console Overview
- **Release Date**: November 12, 2020 (select regions), November 19, 2020 (worldwide)
- **Active Years**: 2020-present
- **Regional Variants**:
  - PS5 Standard Edition (with disc drive)
  - PS5 Digital Edition (digital-only)
  - Minimal regional hardware differences

## Storage Media
- **Disc Capacity**: 100GB Ultra HD Blu-ray
- **Internal Storage**: 825GB custom NVMe SSD
- **Expandable Storage**: M.2 NVMe SSD expansion slot
- **Digital Distribution**: PlayStation Store (primary method)

## Archival Storage
### Current Limitations
- **Emulation**: No emulators exist
- **Dumping**: Limited homebrew/dumping tools
- **Encryption**: Heavy system-level encryption
- **Legal Issues**: Current generation console

### Future Considerations
- Game sizes typically 50-100GB+
- Digital-only titles at risk of loss
- System software preservation important
- Save data cloud backup available

## Digital Storage Considerations
- **Space Requirements**: Enormous - 100GB+ per game
- **Backup Strategy**: Focus on digital library documentation
- **Legal Status**: Current generation - preservation legally complex
- **Future Planning**: Methods and formats still undetermined
- **Priority**: Document and preserve what's possible legally
